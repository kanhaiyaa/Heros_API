To clone and run this code in your machine, you'll need Git, Python v3.8 or higher, Django v3.1.5 or higher and Django Rest Framework v3.12.2 or higher installed on your computer. From your command line:

# Clone this repository.
git clone https://github.com/kanhaiyaa/Heros_API.git

cd Heros_API

# Install dependencies
pip3 install requirements.txt

# Run

python3 manage.py migrate

python3 manage.py runserver

Follow url: http://127.0.0.1:8000/
